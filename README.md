##To Do List

###Developers
Jill Kuchman & Brian Kropff

###Date
April 16, 2015

###Description
This app allows the user to add To Do Lists under various categories.

###Use and Editing
You can use and/or edit this application by:<br />
1. Cloning this project to the directory of your choice<br />
2. Opening the project in your favorite text-editor (optional)<br />
3. Open and point your browser to todo.html<br />
*(ex: file:///project-directory/todo.html)*
